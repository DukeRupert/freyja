# Storefront Overview

Understanding your customer-facing store.

## What Customers See

### Home Page
- Featured products
- Introduction to your roastery
- Quick navigation to shop

### Product Listing
- All available products
- Filter/sort options
- Quick add to cart
- Pricing based on customer's price list

### Product Detail
- Full product information
- Coffee attributes:
  - Origin and region
  - Roast level
  - Tasting notes
  - Process method
- Variant selection (size, grind)
- Add to cart
- Subscription option (if enabled)

### Navigation
- Shop all products
- Account login/signup
- Cart with item count

## Price Display

Prices shown depend on customer status:

| Customer Type | Prices Shown |
|---------------|-------------|
| Anonymous | Retail prices |
| Retail account | Retail prices |
| Wholesale account | Assigned price list |

Wholesale customers only see their tier's pricing.

## Product Visibility

Products appear based on visibility setting:

| Setting | Who Sees |
|---------|----------|
| Public | Everyone |
| Wholesale Only | Wholesale accounts only |
| Hidden | No one |

## Inventory Display

Out-of-stock items:
- Show "Out of Stock" label
- Cannot be added to cart
- Still visible on storefront

## Mobile Experience

The storefront is responsive:
- Works on phones and tablets
- Touch-friendly interface
- Optimized checkout on mobile

## Performance

Storefront is built for speed:
- Server-rendered pages
- Minimal JavaScript
- Fast page loads
- Quick checkout
