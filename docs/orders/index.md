## Overview

Orders in Freyja flow through a standard lifecycle:

```
Pending → Paid → Processing → Shipped → Delivered
```

The admin dashboard provides tools to manage orders at each stage.

## Viewing Orders

1. Go to **Orders** in the sidebar
2. See all orders with status filters
3. Click an order for details

## Order Sources

Orders come from:

- **Storefront checkout** - Customer purchases on your website
- **Subscription renewals** - Automatic recurring orders
- **Wholesale orders** - B2B customer purchases

All orders appear in the same list, filterable by source and status.

## Quick Actions

- **View order details:** Orders > Click order number
- **Update status:** Order detail > Change status
- **Create shipping label:** Order detail > Create Label
- **Filter by status:** Orders > Status filter

## Key Information

Each order shows:

- Order number and date
- Customer name and contact
- Shipping address
- Line items with quantities
- Order total
- Current status
- Shipping tracking (once shipped)
