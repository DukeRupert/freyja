# Go Live Checklist

Use this checklist before launching your Freyja store to ensure everything is configured correctly.

## Payment Setup

- [ ] Stripe account connected
- [ ] Stripe switched from test mode to live mode
- [ ] Business verification completed in Stripe
- [ ] Bank account connected for payouts
- [ ] Test transaction processed successfully

## Products

- [ ] All products created with accurate information
- [ ] Coffee attributes filled in (origin, roast level, tasting notes)
- [ ] SKU variants created for each weight/grind option
- [ ] Product images uploaded (quality photos)
- [ ] Inventory counts set correctly
- [ ] Product visibility set appropriately

## Pricing

- [ ] Retail price list configured
- [ ] Prices set for all products in retail price list
- [ ] Wholesale price lists created (if applicable)
- [ ] Wholesale prices set for all products

## Shipping

- [ ] Shipping provider configured (Flat Rate or EasyPost)
- [ ] Shipping rates verified as accurate
- [ ] Ship-from address set correctly
- [ ] Test shipment created and label generated

## Tax

- [ ] Tax calculation method selected
- [ ] Tax rates configured (if using percentage method)
- [ ] Tax calculation verified on test orders

## Email

- [ ] Email provider configured (Postmark or SMTP)
- [ ] Sender email address set
- [ ] Test emails sending successfully
- [ ] Email templates reviewed

## Checkout Flow

- [ ] Complete checkout tested (cart through confirmation)
- [ ] Order confirmation email received
- [ ] Order appears in admin dashboard
- [ ] Address validation working (if using EasyPost)

## Subscriptions (if offering)

- [ ] Subscription plans created
- [ ] Subscription checkout tested
- [ ] Subscription confirmation email received
- [ ] Stripe Billing subscription visible in Stripe Dashboard

## Wholesale (if offering)

- [ ] Wholesale price lists created
- [ ] Wholesale application form working
- [ ] Invoice creation tested
- [ ] Invoice payment flow verified

## Final Checks

- [ ] All test orders cleaned up or marked as test
- [ ] Store URL correct and accessible
- [ ] Mobile experience tested
- [ ] Contact information accurate
- [ ] Return/refund policy clear

## Launch

Once all items are checked:

1. Switch Stripe to live mode
2. Verify one more live test transaction (small amount, refund after)
3. Announce your launch!

---

## Post-Launch

In the first few days after launch:

- Monitor orders closely for any issues
- Watch for failed payments or checkout errors
- Respond quickly to customer questions
- Check that emails are being delivered
