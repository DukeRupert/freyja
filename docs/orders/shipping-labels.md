# Shipping Labels

Create and print shipping labels for your orders.

## Shipping Providers

Freyja supports two shipping provider options:

### Flat Rate
Simple fixed-price shipping:
- Standard: $7.95 (5-7 business days)
- Express: $14.95 (2-3 business days)

You purchase labels through your own carrier account.

### EasyPost
Real-time carrier integration:
- Compare rates from USPS, UPS, FedEx, DHL
- Purchase discounted labels directly
- Automatic tracking updates

## Creating a Label (EasyPost)

1. Open the order
2. Click **Create Shipping Label**
3. Verify the shipping address
4. Review available shipping rates
5. Select your preferred carrier/service
6. Click **Purchase Label**
7. Print the label

## Printing Labels

After purchase:
1. Click **Print Label** or **Download Label**
2. Print on label paper or regular paper
3. Affix securely to package

### Label Formats
- PDF format (4x6 inches standard)
- Compatible with thermal label printers
- Can print on regular paper and tape to package

## Tracking

When a label is created:
- Tracking number is automatically recorded
- Customer receives shipping confirmation email
- Tracking link included in email
- Order status updates to **Shipped**

## Voiding Labels

If you need to cancel a label:
1. Open the order
2. Find the shipment section
3. Click **Void Label**
4. Confirm void request

Voided labels are refunded to your account (EasyPost).

## Package Specifications

When creating labels, you may need to specify:

| Field | Description |
|-------|-------------|
| Weight | Package weight in ounces or pounds |
| Length | Package length in inches |
| Width | Package width in inches |
| Height | Package height in inches |

### Common Coffee Shipping Sizes

| Contents | Typical Dimensions | Weight |
|----------|-------------------|--------|
| 1 x 12oz bag | 8x6x4" | ~14 oz |
| 2 x 12oz bags | 8x6x6" | ~1.5 lb |
| 1 x 2lb bag | 10x8x4" | ~2.5 lb |
| 4 x 12oz bags | 10x8x6" | ~3.5 lb |

## Best Practices

- **Verify addresses** before purchasing labels
- **Check weights** to avoid carrier adjustments
- **Save common dimensions** for quick entry
- **Print same day** when possible to meet service commitments

## Troubleshooting

### Address Validation Failed
- Check for typos
- Verify apartment/unit numbers
- Contact customer for clarification

### Rate Unavailable
- Check if destination is served by carrier
- Verify package dimensions aren't exceeded
- Try alternative carriers

### Label Won't Print
- Check printer connection
- Try downloading PDF and printing manually
- Verify paper/label stock is loaded correctly
